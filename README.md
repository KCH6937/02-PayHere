## 🙌 실행

### 설치

```
$ https://github.com/team-B-free/02-PayHere.git
$ cd 03-GameDuo
$ npm install
```

### .env 예시

```
NODE_ENV = development
DB_USER = userName
DB_PASSWORD = password
DB_NAME = dbName
DB_HOST = dbHost
DB_DIALECT = mysql
REDIS_PORT = 6397
JWT_SECRET = jwt_secret
```

### 실행

```
$ npm run prepare   //husky 설치
$ npm start
```
